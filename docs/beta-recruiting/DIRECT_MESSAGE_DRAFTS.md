# Direct-message drafts (short form, for texting someone Casey knows)

Produced in an earlier multi-model round-table (`git-native-agents`
`rounds/recruiting-message/`), independently by kimi and GLM, given the
prompt "write the actual first-contact message Casey could send a
captain they know." A third model (mmx/MiniMax) then independently
reviewed both without knowing which was which and preferred GLM's
draft, for making the ask directly instead of circling it — see
reasoning below. Recorded here as committed, durable copies; the
working session that originally produced them has since been wiped by
a machine restart, so this file is the only surviving copy.

These are shorter and more casual than `RECRUITING_COLLATERAL.md`'s
forum post and dock flyer — meant for a direct text or DM to one real
person, not a public post. Fill `[name]` before sending. Per
`docs/FINISH_THIS_PHASE.md`, add a time-bound hook ("looking for 3-4
captains over the next two weeks") before actual use — neither draft
below has one yet.

## Draft A (kimi)

> Hey [name] — I'm helping beta test a new deck log app called
> deckboss.net. Looking for 3–5 boats to run it for 6–8 weeks on the
> water: one setup session, log once a day in real conditions, and a
> quick daily check-in by text or call. No pressure at all — happy to
> tell you more if it sounds interesting, totally fine if not. -Casey

(54 words)

## Draft B (GLM) — mmx's preferred pick

> Hey [name] — quick one, no pressure either way. There's a field beta
> going at deckboss.net and I'm trying to get 3-5 boats I actually know
> in on it. 6-8 weeks, one setup session, log once a day while you're
> fishing, short call or text daily to check in. That's the whole ask.
> If your season's packed or it's just not your thing, say so — totally
> fine, no awkwardness. Happy to send the page if you want a look.

**mmx's verdict** (`mmx text chat`, run against both drafts blind):
Draft B wins clearly — "A talks around the ask... B makes the ask and
then handles my objections... I'd reply to B... I'd let A sit there
until I forgot about it."
